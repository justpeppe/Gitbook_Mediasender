# Requisiti di sistema

### Accesso Web

**Mediasender** è un servizio cloud accessibile via browser, senza necessità di installazione software.\
Serve un dispositivo con connessione internet stabile per caricare file e gestire campagne.\
I browser supportati sono:

* Google Chrome
* Mozilla Firefox&#x20;
* Microsoft Edge

### Accesso Mobile

È disponibile un’app dedicata per dispositivi Android e iOS, scaricabile dagli store ufficiali.\
L’app consente di creare, inviare e monitorare le campagne comodamente da smartphone.\
Si raccomanda di mantenere il sistema operativo e l’app aggiornati per garantire performance e sicurezza ottimali.

* [Link Android](https://play.google.com/store/apps/details?id=it.xdevel.mediasender\&hl=it\&pli=1)
* [Link iOS](https://apps.apple.com/bb/app/mediasender/id6736950857)

### Requisiti aggiuntivi

I file audio possono essere caricati in formato WAV, con conversione automatica in MP3.\
L’editor delle immagini e tutte le funzionalità di modifica sono operative direttamente in cloud, senza bisogno di altri software.
